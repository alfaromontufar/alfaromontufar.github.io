JGraphs

This version of JGraphs is implemented in javascript, to graphically manipulate graphs. It can be used to generate code for other software, like SageMath, LaTeX, etc. In the past, I have used it to compute the critical ideals and critical groups of graphs, as well as, create Tikz code for LaTeX. Since it was coded in javascript, it works on different browsers.
Basic command for the TextBox:
Kn - Creates a complete graph with n vertices
Pn - Creates a path with n vertices
Tn - Creates a graph with no edges and n vertices
It also can read the adjacency matrix of a graph
Examples:
Try K10 followed by ENTER.
Try {{0,1},{0,0}} to produce an arrow; it also works P2.

Sandpile model 1

Randomly one grain is added to one node. If the obtained configuration is unstable, then it is toppled until it get stable. So on.
Here gray-scale is used to represent that the number of grains are in a node. Black color means no grain is in this node. And white means that more than 3 grains are in the node (unstable)

In this model the central vertex has 1000 grains. It topples until stabilizes.
